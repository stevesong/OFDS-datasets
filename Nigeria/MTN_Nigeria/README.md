# MTN Nigeria #

This map was digitized from the publicly available MTN Nigeria\ network map at https://www.mtn.ng/wp-content/uploads/2022/11/MTN_Nigeria_Investor_Presentation-April_2018.pdf (page 68). It is not a perfect transcription as some routes and locations are hard to make out and/or geo-locate.  Routes are approximations.
