# Vocus Australia #

This map was digitized from the publicly available Vocus network map at <https://www.vocus.com.au/about-vocus/our-network/>. Routes on this map are guesses based on road infrastructure between Points of Presence. This is a useful map to give a sense of the bigger picture but should not be treated as in any way authoritative.
