# Camtel Cameroon #

This map was digitized from the publicly available the Camtel Carrier map at https://carrier.camtel.cm/coverage. It is not a perfect transcription as some routes and locations are hard to make out and/or geo-locate. Routes are approximations.

-- Steve Song, November 2025
