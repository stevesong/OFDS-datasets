# National ICT Broadband Backbone (NICTBB) #

This map was digitized from the publicly available map posted to the [TTCL](https://www.ttcl.co.tz/) instagram on 26 July 2025 <https://www.instagram.com/p/DMkvYZKNeNs/>. Routes on this map are guesses based roughly on driving directions from PoP to PoP. This is a useful map to give a sense of the bigger picture but should not be treated as in any way accurate.
