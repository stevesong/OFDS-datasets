# Georgia Open Net #

This map was converted to OFDS from the publicly available, downloadable, KML map at <hhttps://maps.opennet.ge/en/>. Conversion was done with the kml2ofds script at <https://github.com/Open-Telecoms-Data/kml2ofds>.  Area polygons in the KML were removed first.
