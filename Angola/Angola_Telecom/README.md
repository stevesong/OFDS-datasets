# Angola Telecom #

This map was digitized from the publicly available Angola Telecom network map at https://www.fiberekci.com/kutuphane/angola-telecom-angola/. It is not a perfect transcription as some routes and locations are hard to make out and/or geo-locate.  Routes are approximations.
