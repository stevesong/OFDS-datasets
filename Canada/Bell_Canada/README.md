# Bell Canada#

This map was digitized from the publicly available Bell Canada map at <https://wholesale.bell.ca/Portals/0/pdf/Bell_Network_Map_2024.pdf>. Routes on this map are guesses based on Google driving directions from PoP to PoP. This is a useful map to give a sense of the bigger picture but should not be treated as in any way accurate.
