# Dempster Fibre Line #

This map was digitized from the publicly available map at <https://yukon.ca/en/dempster-fibre-line-project#Projectstatus>. Routes on this map are guesses based on Google driving directions from PoP to PoP. This is a useful map to give a sense of the bigger picture but should not be treated as in any way accurate.
