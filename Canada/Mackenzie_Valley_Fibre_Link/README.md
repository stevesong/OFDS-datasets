# Mackenzie Valley Fibre Link #

This map was digitized from the publicly available map at <https://www.mvfl.ca/area-map/>. Routes on this map are guesses based on driving directions from PoP to PoP. This is a useful map to give a sense of the bigger picture but should not be treated as in any way accurate.
