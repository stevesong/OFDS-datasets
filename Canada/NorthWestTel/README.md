# NorthWestTel #

This map was digitized from the publicly available NorthWestTel map at <https://www.nwtel.ca/community/canada-north-fibre-loop>. Routes on this map are guesses based on Google driving directions from PoP to PoP. This is a useful map to give a sense of the bigger picture but should not be treated as in any way accurate.
