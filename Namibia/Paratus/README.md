# Paratus Namibia #

This map was digitized from the publicly available Paratus Namibia network map at https://paratus.africa/namibia/wp-content/uploads/2025/08/ParatusNamibia_CoverageMap.webp. It is not a perfect transcription as some routes and locations are hard to make out and/or geo-locate.  Routes are approximations.
