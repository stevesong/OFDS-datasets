# Chorus New Zealand #

This map was digitized from the publicly available Chorus datacentres map at <https://www.chorus.co.nz/enterprise/data-centre-connectivity>. Routes on this map are guesses based on road infrastructure between Points of Presence. This is a useful map to give a sense of the bigger picture but should not be treated as in any way authoritative.
